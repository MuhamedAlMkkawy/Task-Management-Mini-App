# Task Management Mini App

## Getting Started

Follow these steps to run the project locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/MuhamedAlMkkawy/Task-Management-Mini-App.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Task-Management-Mini-App
   ```

3. Install the project dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open the local URL displayed in your terminal (typically `http://localhost:3000` or `http://localhost:5173`, depending on your setup).
